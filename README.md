# enrapt.io

Marketing/personal website for Fabian Alsultany, Strategic Advisor, Fractional COO, Business Development Executive, under the ENRAPT brand.

This is a **static site**: a single self-contained index.html (fonts, images, logo, and styles are all embedded). It is hosted on **GitHub Pages** at **https://enrapt.io**.

## Files

| File | Purpose |
|------|---------|
| index.html | The entire website. This is what gets served. |
| CNAME | Tells GitHub Pages the custom domain is enrapt.io. Do not delete. |
| README.md | This file. |

## How it's deployed
GitHub Pages Settings - Pages - Source: main branch, root - Custom domain: enrapt.io - Enforce HTTPS.

## How to update the site
Small text edits: open index.html here on github.com, click the pencil (Edit), make the change, Commit. Live in ~1 minute.
Bigger changes: replace index.html via Add file - Upload files (overwrite), then Commit.
No build step: what's committed is what's live.
